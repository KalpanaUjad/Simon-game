# Simon Game 

This is a simple Simon Game built using HTML, CSS, and JavaScript.  
It tests the user's memory by flashing colored buttons in a specific sequence that the user must repeat.

##  Features
- Interactive gameplay
- Colorful design
- Sound effects for each button

##  Tech Stack
- HTML
- CSS
- JavaScript

##  How to Play
1. Press any key to start the game.
2. Repeat the sequence of colors.
3. Each level adds a new color to the sequence.
4. If you make a mistake, the game restarts.
